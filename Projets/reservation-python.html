Projet 05 - Développement logiciel

Développement d’un système de réservation de billets avec Python

Application développée avec Python dans le cadre d’un projet logiciel géré avec une approche Agile/Scrum.

Objectif du projet

Développer un système permettant la réservation de billets tout en appliquant une méthode de gestion de projet Agile pour organiser le développement, suivre les tâches et livrer une application fonctionnelle.

Architecture utilisée

  • Application développée en Python.
  • Structure organisée en modules ou fichiers de logique applicative.
  • Gestion des fonctionnalités selon des tâches Agile.
  • Dépôt GitHub pour conserver le code source.

Outils utilisés

  • Python
  • GitHub
  • Méthode Scrum Agile
  • Planification des tâches

Étapes réalisées

  1. Définition du besoin fonctionnel.
  2. Identification des fonctionnalités principales.
  3. Planification du travail selon une approche Agile.
  4. Développement des fonctions de réservation.
  5. Tests fonctionnels de l’application.
  6. Correction des erreurs identifiées.
  7. Publication du projet sur GitHub.

  • Application de réservation fonctionnelle.
